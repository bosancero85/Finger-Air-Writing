# Finger Air Writing mit Python

Dieses Programm nutzt OpenCV und MediaPipe, um Handbewegungen über die Webcam zu verfolgen und virtuell auf dem Bildschirm zu schreiben.

## Features
- **Zeichnen:** Nur der Zeigefinger ist oben.
- **Auswahlmodus/Pause:** Zeige- und Mittelfinger sind oben.
- **Löschen:** Im Auswahlmodus die Hand in die obere linke Ecke bewegen.

## Installation
1. Repository klonen
2. Virtuelle Umgebung erstellen: `python -m venv venv`
3. Aktivieren: `venv\Scripts\activate` (Windows)
4. Bibliotheken installieren: `pip install -r requirements.txt`

## Starten
Führe das Skript mit `python script.py` aus.
